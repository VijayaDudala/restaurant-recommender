# 🧠 Machine Learning Used

This project uses a recommendation system based on:

* Cosine Similarity
* User-item interaction matrix
* Collaborative Filtering

Libraries Used:

* Pandas
* Scikit-learn
* NumPy

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/VijayaDudala/restaurant-recommender.git
```

---

## 2️⃣ Backend Setup

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r ../requirements.txt
python manage.py migrate
python manage.py runserver
```

Backend runs at:

```bash
http://127.0.0.1:8000/
```

---

## 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```

Frontend runs at:

```bash
http://localhost:3000/
```
